# Changelog

## v1.2.0 - April 24, 2026

- Generated lineups now auto-save immediately so you do not lose the game plan.
- Saved games clearly show which ones still need innings played before season stats are finalized.
- Changing innings or innings actually played now updates the saved draft automatically.

## v1.1.0 - April 19, 2026

- Added visible app version labels in the app and on printable packets.
- Moved version info into a dedicated release file so app displays stay in sync.
- Improved offline cache handling so updated builds show up more reliably.
